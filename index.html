Quick Start

Get up and running in minutes. Download the dataset from Kaggle and place it in the data/ folder.

Installation

pip install -r requirements.txt

Prepare Data

python main.py --mode prepare-data

Train Model

python main.py --mode train-model

Evaluate Model

python main.py --mode score-model

Dataset

The dataset uses IOB format (Inside-Outside-Beginning) for entity tagging. Each token is labeled with its entity type.

Tag Format

  • B- Beginning of an entity
  • I- Inside/continuation of an entity
  • O Outside any entity

Parameters

Built on RoBERTa-base with a linear classification head for token-level entity prediction. The model dynamically loads the hidden size from the pretrained configuration.

Training Configuration

data:
  path: './data/'
  wandb: 'entity-extraction'

params:
  device: 'cuda:0'
  hfmodel: 'roberta-base'
  bsize: 12
  learning_rate: 1e-5
  epochs: 2

score:
  model: 'model.pt-v1.ckpt'

Key Components

  • Loss: Cross-entropy with padding token masking
  • Optimizer: AdamW
  • Framework: PyTorch Lightning 2.0+
  • Logging: Weights & Biases
